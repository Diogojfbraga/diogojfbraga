# Hi, 👋

I’m a Data Information Manager with a clinical background (critical care) transitioning into cybersecurity.  
I build data-driven security projects—especially detection and ML-based intrusion detection.

## What I’m focused on
- Security analytics & threat detection (SIEM-style pipelines, log analysis)
- Machine learning for IDS (supervised models, evaluation, performance trade-offs)
- Python, SQL, and practical security tooling (scans, parsing, automation)

## Featured projects
- **ML Intrusion Detection System (CICIDS2017)** — Random Forest, SVM, CNN/RNN; preprocessing + evaluation (accuracy/F1/time)
- **SQL Analytics (Healthcare Ops)** — performance-focused queries, reporting pipelines, dashboards
- **CTF/Blue Team Notes** — repeatable notes + commands for enumeration, forensics, and detection

## Education
